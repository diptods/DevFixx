1. Create React App (Vite)
npm create vite@latest devfixx -- --template react
cd devfixx
npm install
2. Install Routing
npm install react-router-dom
3. Tailwind CSS
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
4. Animation Tools
npm install framer-motion gsap
5. Smooth Scroll
npm install @studio-freight/lenis
6. Icons
npm install lucide-react
7. Forms
npm install react-hook-form zod
8. API Handling
npm install axios
9. Supabase (Backend-as-a-Service)
npm install @supabase/supabase-js
10. Email Service (Optional but PRO)
npm install resend
11. Analytics (Optional)
npm install posthog-js
🚀 AFTER SETUP (IMPORTANT STEPS)
1. Initialize Git
git init
git add .
git commit -m "DevFixx initial setup"
2. Push to GitHub
git branch -M main
git remote add origin https://github.com/yourname/devfixx-website.git
git push -u origin main
3. Deploy
